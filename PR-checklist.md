# Pull Request Checklist (New Members)

Before submitting your Pull Request, confirm the following:

## Repository Setup

* [ ] I forked the `openiqx-members` repository
* [ ] I cloned my fork locally
* [ ] I created a feature branch

Example:

```
git checkout -b feat/add-your-name
```

---

## Profile File

* [ ] I created my file inside the `members/` directory
* [ ] The filename follows the correct format

Example:

```
john-doe.md
```

* [ ] I used the provided template

---

## File Content

* [ ] My name is included
* [ ] My GitHub link is included
* [ ] My LinkedIn link is included
* [ ] Programming languages are listed

---

## Contribution Rules

* [ ] Only **merged pull requests** are listed
* [ ] Each contribution includes both a **Pull Request** link
* [ ] The format matches the example provided

---

## Git Workflow

* [ ] I committed my change

```
git add .
git commit -m "Add profile for Your Name"
```

* [ ] I pushed my branch

```
git push origin feat/add-your-name
```

---

## Final Step

* [ ] I opened a Pull Request to the `main` branch

Once your Pull Request is **reviewed and merged**, you are officially an **OpenIQX community member**.
